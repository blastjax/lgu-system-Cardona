# lgu-system-cardona

## Setup
- Create ```.env``` file (check ```.env.sample``` for reference)
- Install dependencies from requirements.txt: ```pip install -r requirements.txt```
- Run migrations: ```alembic upgrade head```
- Run command: ```uvicorn app.main:app --reload```
